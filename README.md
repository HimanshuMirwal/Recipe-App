🍽️ Recipe App
This is a basic recipe app built using React Native with Expo. It displays a variety of cuisine categories like Italian, German, and more. Users can explore recipes by category and mark their favorites — all without the use of a database.

📱 Features
  - 🧭 Navigation using stack & drawer Navigators
  - 🥘 Browse recipes by cuisine categories
  - 📋 View details of individual recipes
  - ❤️ Add and remove recipes from favorites
  - ⚙️ Built entirely with React Native + Expo
  - 🗃️ No backend or database — purely frontend logic

🧩 Tech Stack
  - Expo
  - React Native
  - TypeScript

🚀 Getting Started

1. Clone the repo
   git clone https://github.com/your-username/recipe-app.git
2. cd recipe-app
   npm install
   # or
   yarn install
3. Start the app
   npx expo start
Open in Expo Go or an emulator.

⚠️ Notes
  - This app does not use a backend — all data is static or locally handled.
  - Favorites are not persisted across app reloads (no AsyncStorage or database).
  - Built for educational and prototyping purposes.
